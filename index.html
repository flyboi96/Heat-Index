Heat Index

Track strength, conditioning, and performance. Log PRs anytime. Build your score. Compete in the heat.

Challenge Overview

  1. Test five events: squat, bench, event 3, farmer hold, and the 1.64-mile run.
  2. Men use strict pull-ups for Event 3. Women use dead-bar hang time.
  3. Each event is worth 20 percent. Strong across all five beats being elite at just one.
  4. No single test day. Log a PR whenever you earn one.

How to Compete

  1. Open the Score tab.
  2. Select your last name, then enter only the PRs you want to upload.
  3. Tap Update Score to preview your Heat Index total.
  4. Tap Upload PRs to save new numbers. Blank fields keep your old PRs.

How Scoring Works

  1. Each event scores from 0 to 100 based on where you fall between the tier standards.
  2. Beginner = 20, Novice = 40, Intermediate = 60, Advanced = 80, Elite = 100.
  3. You can score above Elite, so there is always room to improve.
  4. Your final Heat Index score is the average of all five event scores.

Harry Squatter, 155 lb

Challenge Rules

  1. Strength events (squat and bench) are scored as 3RM divided by bodyweight.
  2. Event 3: men perform strict pull-ups, women perform a dead-bar hang.
  3. Farmer hold: 2 x 24 kg / 53 lb (men), 2 x 16 kg / 35 lb (women).
  4. Run: 1.64 miles total. One out-and-back (0.82) completed twice.
  5. Each event is worth 20 points per tier: Beginner 20, Novice 40, Intermediate 60, Advanced 80, Elite 100.
  6. No ceiling. Scores continue past Elite. Keep pushing.

What Counts

  1. Squat: below parallel, controlled, and clearly completed for 3 reps.
  2. Bench: bar touches chest, full lockout, and no excessive bounce.
  3. Pull-ups: dead hang to chin clearly over bar. No kipping.
  4. Dead-bar hang: full grip, feet off ground, no touching down.
  5. Farmer hold: stand tall, no resting on legs, ends when one bell touches down.

Run Route

Start at the marked start/finish point, run to the turnaround point, then return to start. That out-and-back is 0.82 miles total. Repeat the same out-and-back one more time for 1.64 miles total. The route is just south of the soccer field/NEX.
